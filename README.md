# SimpleKeyLogger
Simple KeyLogger made with Python for cybersecurity. 

It has screnshoot capabilities as well as extraction of data collected via email.

## 2) Dependencies

* Pynput 
* Pywin32
* Pillow

## 3) Python version used for development
